Projects

DrySmart+ Rain Safe Drying System

This hardware project focuses on providing a smart solution for drying clothes efficiently, even in unpredictable weather conditions. It incorporates sensors to detect rainfall and automatically retracts the drying system to protect clothes. The system is designed to be energy-efficient, user-friendly, and customizable based on different drying needs. This project emphasizes innovation and sustainability by reducing dependency on traditional drying methods.

NutriWell:Personalized Nutrition Planner(Web Application)

This project is a web application that provides users with a personalized nutrition plan based on their age, weight, dietary preferences, and health goals. NutriWell incorporates a simple, interactive interface and uses algorithms to recommend meals and track users' progress. The system is designed to promote healthier eating habits, offering recipe suggestions and nutrient breakdowns while ensuring users achieve their health objectives. EduBot:AI-Powered Study Assistant(Software Project)

EduBot is an AI-driven virtual assistant designed to help students organize their studies more effectively. The software includes features like customizable study schedules, reminders, progress tracking, and access to curated educational resources. It also incorporates a chatbot that can answer basic academic questions and provide motivational prompts to keep users focused and productive. This project highlights the integration of technology into modern education to enhance learning experiences.
